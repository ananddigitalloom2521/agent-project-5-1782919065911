# Vacation Request App
A simple application for requesting and approving vacation time.
## Getting Started
1. Clone the repository: `git clone https://github.com/your-username/vacation-request-app.git`
2. Install dependencies: `npm install`
3. Create a `.env` file and add your MongoDB URI and port number
4. Start the server: `npm start`
## API Endpoints
* `GET /api/users`: Get all users
* `POST /api/users`: Create a new user
* `GET /api/vacation-requests`: Get all vacation requests
* `POST /api/vacation-requests`: Create a new vacation request
* `PUT /api/vacation-requests/:id`: Update a vacation request
